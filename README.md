SPECTRE VIS-OPS Center

Tactical Field Interface | Loyal Serpent 8 (Regina)

SPECTRE VIS-OPS is a specialized, offline-first Progressive Web App (PWA) designed for the Spectre squad during the Loyal Serpent 8 Milsim event. It provides a centralized hub for mission intelligence, squad logistics, and emergency medical data.
